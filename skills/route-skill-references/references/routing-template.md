# Routing Template

Use this shape in `SKILL.md`:

```md
## Reference Routing

- Topic A: `references/topic-a.md`
- Topic B: `references/topic-b.md`
- General source links: `references/official-sources.md`
```

Rule:

- Only open the reference that matches the user’s requested topic.
- If the topic is not covered locally, consult the official source.
