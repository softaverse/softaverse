# Mermaid for Obsidian

Use this skill to create Mermaid diagrams in Obsidian.

## How To Use This Skill

1. Identify the diagram topic first.
2. Read only the matching file in `references/`.
3. Use the examples and syntax in that file to answer or generate the diagram.
4. If the needed usage is not covered there, consult `references/official-sources.md`.

## Reference Routing

- Flowchart or graph syntax: `references/flowchart.md`
- Sequence diagrams: `references/sequence.md`
- Class diagrams: `references/class.md`
- Gantt charts: `references/gantt.md`
- State diagrams: `references/state.md`
- Pie charts: `references/pie.md`
- General source links: `references/official-sources.md`

## Rule

Do not display all references at once. Only open and use the reference that matches the user’s requested Mermaid topic.
